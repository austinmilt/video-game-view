Class: AvatarBox

AvatarBox(positionopt)

Class for UI elements that are Avatar boxes. AvatarBox's are very dynamic and contain info from multiple other boxes (i.e. everything in AbilityBox, ItemBox, and TalenBox), so they are a bit more complex.
